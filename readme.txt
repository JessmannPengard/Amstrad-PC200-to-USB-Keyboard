1. Copiar flash_nuke.uf2 a la raíz para limpiar la memoria flash (opcional).
2. Copiar adafruit-circuitpython-raspberry_pi_pico-es-9.1.3.uf2 a la raíz.
3. Copiar la carpeta kmk a la raíz.
4. Copiar boot.py a la raíz para evitar que se inicie en modo almacenamiento masivo.
5. Copiar main.py a la raíz.
6. Desconectar y volver a conectar la Pi Pico.

* Scanner.py permite identificar a qué fila y columna de la matriz pertenece una tecla al ser pulsada. Útil para identificar las teclas y mapearlas.
